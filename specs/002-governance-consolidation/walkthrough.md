# Walkthrough — Консолидация governance (RULES.md + AGENTS.md + полная дизолюция)

Отчёт о реализации фичи `002-governance-consolidation`. Исполнители: Claude Opus (Chunk 1: Фазы 1–4), Claude Sonnet (Chunk 2: Фазы 5–7), через `speckit-implement`.

## Changes made (Внесённые изменения)

**Единый рулбук `RULES.md` (публичный, tracked):** 60 правил в 9 доменах (ARCH, DB, UI, FSM, DATA, CODE, TEST, PROC, SEC), у каждого — ID `R-<DOMAIN>-<n>`, формулировка, Why, tier (A — суждение / B — CI-enforced), legacy-якоря. 16 групп дубликатов слиты в одиночные правила по политике «строже побеждает» (например: `PL-6.4↔CP-3.19→R-UI-1`, `CP-3.29↔GEMINI§GW-1→R-PROC-5`). 14 Tier-B правил понижены до однострочников с проверяемыми указателями на enforcement.

**Конституция `AGENTS.md` (переведена из игнорируемого реестра субагентов в трекаемую конституцию):** идентичность, онбординг, маршруты A–D, реестр команд, RNA-Blueprint, git-протокол, протокол ответов, реестр файлов, content ownership, субагенты — цитирует R-ID, не копирует текст правил. `CLAUDE.md`/`GEMINI.md` стали чистыми шимами-указателями.

**`docs/knowledge/rule-map.md`:** 295 legacy-якорей (`PL-x.y`/`CP-x.y`) резолвятся к R-ID или файлам бандла — ни одна историческая цитата не потеряна.

**Полная дизолюция:** 7 новых файлов бандла созданы по карте research.md D6 дословным переносом описательного контента — `architecture.md`, `middleware.md`, `fsm-protocol.md` (включая механику UIService, не покрытую исходной картой D6 — пробел закрыт прагматично), `db-patterns.md`, `constants.md`, `testing.md`, `features-overview.md`. `PROJECT_LOGIC.md` и `CONTEXT_PROMPT.md` переписаны в тонкие редирект-индексы (≤15 строк каждый) — теперь буквально то, чем и должен был быть `PROJECT_LOGIC.md` по замыслу Шэфа: указатель, а не хранилище смешанного контента.

**Синхронизация инструментов:** `tenirtoo-docs-update` — producer-контракт v2 (правила → RULES.md, справка → бандл, процесс → AGENTS.md, правило неизменности ID); `tenirtoo-proposal-analysis` — ground truth заменён с `PROJECT_LOGIC.md` на `RULES.md` + `docs/knowledge/` (7 мест правки). Spec-kit constitution заполнена принципами верхнего уровня.

**Граф знаний перестроен** (`graphify --update`, инкрементально): 1195 узлов, 2536 рёбер, 135 сообществ (было 1002/2309/66) — здоровье графа ОК, диффа (247 новых узлов, 54 удалённых) точно отражает замену старой структуры на новую.

Исходный код бота **не изменялся**.
